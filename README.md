# progressive-digital/drupal-patches

Eingefrorene Patch-Dateien der PROGRESSIVE Drupal-Distribution. **Öffentlich**, weil alle Patches
von öffentlichen drupal.org-/git.drupalcode.org-Issues stammen — hier nur eingefroren, damit gelöschte
Issue-Forks oder bewegte MR-Diffs keine Builds brechen.

Konsumiert vom (privaten) Fleet-Metapaket `progressive/base` via `raw.githubusercontent.com`-URLs
(composer-patches v2 Dependency-Patches). Verdikte/Herkunft je Patch: `PATCHES.md` im Distro-Template.

**Nicht direkt bearbeiten** — Patches werden über den Distro-Wartungsprozess erneuert (Reroll → neuer Commit/Tag).
